# MagicPic 基于express+nodejs的壁纸站。

> 技术栈：express + nodejs + mongoose

## 使用说明

## Environment

`Node >= 6`

`mongoDB`

``` bash
#克隆项目
git clone git@github.com:magictear/magicpic.git

# 安装依赖
cd MagicPic
npm install

# 构建
npm start

# 本地开发环境 访问http://localhost:3000

```

## 功能

*背景滚动首页
*注册、登录页面
*主页面
*壁纸分类页面
*用户页面
*自定义添加分类功能
*壁纸预览
*壁纸详情页面
*添加本地壁纸功能
*编辑壁纸功能
*帅选、搜索功能
*其他页面
